HOW WE CLONE A CHOSEN SITE

1. You (or they) pick one reference site from the list (or send another URL).
2. We keep: Riverside name, logo, colors (green/orange), content (fish, fowls, Director, contact).
3. We copy from the reference: layout (sections, hero style, card layout), spacing, button style, and “vibe” (minimal, glassy, farm-story, etc.).
4. We do not copy: their text, images, or branding—only structure and style.
5. Build time: once the reference is chosen, cloning the structure usually takes a few hours to a day depending on how close they want it.

Result: A site that feels like the reference they chose but 100% Riverside Agropro.
